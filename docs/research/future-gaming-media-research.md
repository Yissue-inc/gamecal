## 1. API/크롤링 난이도 기준

| 난이도 | 설명 | 방법 |
|--------|------|------|
| ✅ 쉬움 | RSS 피드 공식 제공 또는 공식 API 존재 | RSS 파싱, 공식 API Key |
| ⚠️ 중간 | JS 렌더링 필요 또는 비공식 API 활용 | Playwright/Puppeteer, 비공식 엔드포인트 |
| ❌ 어려움/불가 | Cloudflare 보호, 유료 구독 장벽, 공식 API 없음 | 우회 거의 불가 또는 유료 서비스 필요 |

## 2. 🇺🇸 미국 주요 게이밍 미디어 20선

> 요약: 공식 API 제공 8개 / 크롤링 가능 9개 / 제한·불가 3개

### 1. IGN
- **URL:** ign.com
- **월 방문자:** 약 286M (2025년 1월 기준)
- **운영사:** IGN Entertainment
- **커버 범위:** 전 플랫폼 종합 (PC, PS, Xbox, Nintendo, Mobile)
- **제공 콘텐츠:** 뉴스, 리뷰, 공략/가이드, 위키, E-스포츠, 영상
- **API/크롤링:** RSS 공식 제공 / 크롤링 가능 (난이도: 중간)
- **비고:** 전 세계 35개 플랫폼, 110개국 배포. 가장 광범위한 게임 미디어.

### 2. GameSpot
- **URL:** gamespot.com
- **운영사:** Red Ventures 소속
- **커버 범위:** PS, Xbox, Nintendo, PC, Google Stadia
- **제공 콘텐츠:** 뉴스, 리뷰, 위키, 커뮤니티, 라이브 스트림
- **API/크롤링:** RSS 공식 제공 / 크롤링 가능 (난이도: 중간)
- **비고:** DA(도메인 어소리티) 92. 1990년대 창립. 업계 최고 권위 리뷰 시스템.

## 7. 어필리에이트 전략 요약

### 목적별 최적 조합

| 목적 | 추천 플랫폼 | 예상 수수료 |
|------|-------------|-------------|
| **초보 입문 / 범용** | Amazon Associates + Fanatical | 5~12% |
| **PC 게임 사이트** | GMG + Fanatical + GOG + Humble 스택 | 5~15% |
| **고단가 수익** | Razer + Secretlab | 12% (건당 $10~84) |
| **스트리머 채널** | Nerd or Die + Epic SAC 코드 | 5~30% |

| **퍼블리셔 프랜차이즈** | Ubisoft 크리에이터 코드 | 7% |

### 지역별 어필리에이트 전략

- **미국/글로벌:** Amazon Associates + Fanatical + GMG 조합 최적
- **유럽:** Gamesplanet (공식 인증 강점) + GOG (DRM-free 유럽 선호)
- **남미:** 브라질은 Mercado Pago 연동 플랫폼 우선. Amazon BR + Nuuvem 고려
- **스페인어권:** Epic SAC (Fortnite 인게임 아이템) + Amazon ES/MX

### 핵심 전환율 인사이트

- 게임 리뷰 페이지: 전환율 1~3% (리서치 단계 독자)
- 딜 알림 / 세일 페이지: 전환율 6~10% (구매 의향 독자)
- 번들 페이지: 전환율 중간 이상 (할인 + 가치 인식)
- **고수익 전략:** 가격 비교 테이블 + 딜 알림 + 번들 전문 콘텐츠

# GamerClock SEO/GEO Promotion Plan

Last updated: 2026-06-21

## Positioning

GamerClock should be described consistently as:

> GamerClock is a gaming event calendar for live events, release dates, weekly resets, esports fixtures, reminders, and ROAR match participation.

ROAR should be described consistently as:

> ROAR is a free Summer Cup 2026 crowd battle game on GamerClock where players pick a side, cheer for a nation, and save rank progress.

Use "Summer Cup 2026" as the generic tournament theme. Do not use official FIFA marks, logos, or affiliation language.

## Big Rocks

1. Own the category language
   - gaming event calendar
   - game release calendar
   - live game events
   - weekly reset tracker
   - esports fixture calendar
   - Summer Cup 2026 fixtures
   - ROAR crowd battle game

2. Build indexable landing pages
   - `/` for the general gaming calendar.
   - `/summer-cup` for fixtures, results, standings, goals, and ROAR momentum.
   - `/summer-cup/[id]` for each match.
   - `/roar` for the playable game.
   - `/games/[slug]` for each game calendar.

3. Make AI answer engines understand the entity
   - Maintain `/llms.txt`.
   - Keep schema.org data accurate.
   - Use consistent names: GamerClock, ROAR, Summer Cup 2026.
   - Publish factual pages with clear summaries, not only app UI.

4. Create repeatable external signals
   - Product Hunt launch.
   - Reddit posts in game/event/calendar communities where rules allow.
   - X/TikTok short clips for ROAR match moments.
   - Indie Hackers/build-in-public updates.
   - GitHub README/public changelog references if open-source assets are used.
   - Directory submissions: PWA directories, startup directories, gaming tool lists.
